## 3. 游戏循环 (按顺序做一遍)

```
   ┌──────────────┐
   │ 📋 申请采购单 │  ← 100 金币换一张订单
   └──────┬───────┘
          ↓
   ┌──────────────┐
   │ 🛒 采购原料  │  ← 用金币买资源 / 本国 L1 / 任何国家的 L2 · L3
   └──────┬───────┘
          ↓
   ┌──────────────┐
   │ 🏭 生产产品  │  ← 丢骰: 1=失败资源浪费; 2-6=成功
   └──────┬───────┘
          ↓
   ┌──────────────┐
   │ 💸 运输销售  │  ← 丢骰: 4=运输失败产品丢; 1/2/3/5/6=赚金币
   └──────┬───────┘
          ↓
   ┌──────────────┐
   │ 🎁 研发投资  │  ← 500 金币, 80% 抽奖, 20% 失败钱没了
   └──────────────┘
```

### Step 1 · 📋 申请采购单

- 找导师按 **📋 订单** 按钮 → 选你要做哪一级别 (L1 / L2 / L3 / L4)
- 花 **100 金币**, 系统**随机**给你一张该级别的产品订单 (从你国家的产品池抽)
- 同时持有采购单上限随国家等级提升: **Lv.1 → 2 张 · Lv.2 → 3 张 · Lv.3 → 5 张 · Lv.4 → 5 张**
- 🔒 **申请级别不能高于国家当前 Level** — Lv.1 只能申请 L1; Lv.2 可申请 L1/L2; Lv.3 可申请 L1/L2/L3; Lv.4 全开 (最高级)。系统会把高于国家等级的按钮锁住 (🔒)
- ⚠️ 没采购单就不能生产

> **为什么是随机?** 模拟真实世界 — 你不能挑客户要哪个产品,得做市场要的那个。

### Step 2 · 🛒 银行采购

- 找导师按 **🛒 采购** 按钮
- 四个 Tab:
  - **基础资源** — 任何国家都能买 6 种资源
  - **L1 产品** — 只能买**本国**的 L1 (基础半成品,自己做就行)
  - **L2 产品** — 可以买**本国 + 其他国家**的 L2 (跨国买高级半成品凑配方)
  - **L3 产品** — 可以买**本国 + 其他国家**的 L3 (用来凑 L4 配方)
- 用金币结算,扣完显示在卡片上
- L2/L3 列表会标出来源国家 (🇲🇾 / 🇰🇷 / 🇯🇵 / 🇺🇸 / 本国)
- 🔒 **采购分类同样受国家等级闸控:**

  | 国家等级 | 可买分类 |
  |---|---|
  | Lv.1 | 仅资源 |
  | Lv.2 | 资源 + L1 |
  | Lv.3 | 资源 + L1 + L2 |
  | Lv.4 (最高) | 资源 + L1 + L2 + L3 |

  超过本国 Level 的 tab 会变成 `🔒 LX 产品` 不可点

> 💡 **跨国谈判仍可行:** 如果想用比银行更便宜/更贵的协商价,仍可走 💸 销售 modal 跟对方国家谈。银行采购是"快速通道",谈判是"灵活通道"。

### Step 3 · 🏭 生产

- 必须先有采购单 (📋), 才能在 🏭 modal 里看到可生产产品
- 系统会显示该产品的配方:
  - 需要多少资源 (红色 = 你不够)
  - 需要多少半成品 (你库存有没有)
- 资源/半成品够了 → 丢骰子:
  - 🎲 **1 = 失败** (资源照样扣, 采购单也消耗, 但拿不到产品)
  - 🎲 **2/3/4/5/6 = 成功** (拿到产品)

> ⚠️ **失败不退资源,采购单也作废。** 失败了就要重新申请采购单。这是真实工业的风险。

### Step 4 · 💸 销售

- 找导师按 **💸 销售** → 选产品 → 选卖给:
  - 🏦 **银行** (固定价,Prices 表里写的) — 任何 L1-L4 都可卖
  - 🌍 **其他国家** (协商价,双方谈) — 仅 L1/L2/L3 (**L4 不能跨国**)
- 输入数量 + 单价 + 原因
- 丢运输骰:
  - 🎲 **4 = 运输失败** (产品损失, 没收到钱)
  - 🎲 **1/2/3/5/6 = 成功** (扣对方/银行金币, 给你金币, shipments +1)
- 如果你有 🎫 **免失败运输卡**, 可以勾选"使用免失败卡"跳过骰子,**自动成功**

> ⚠️ **L4 产品只能本国生产 + 卖给银行 (🏦)。** 不能从银行采购 (🛒 没有 L4 tab),也不能跨国卖给其他国家。你的 L4 库存只用来:(a) 算总资产 (asset_value 高), (b) 卖给银行换金币。

### Step 5 · 🎁 研发投资

- 找导师按 **🎁 研发** → 看奖品池预览 → 投入 **500 金币**
- 20% 概率**失败** (钱全没,啥也没拿到)
- 80% 概率**成功**, 从奖品池随机一种:
  - 📦 **资源包 ×2** (某种资源 +200 单位)
  - 🎫 **免失败运输卡 ×1 或 ×2** (运输时勾选可跳过骰子)
  - 💰 **金币 +1000 / +2000**

> 💡 **什么时候研发?** 金币多于 1600 + 急需某资源/卡时再投。开局别乱研发。

---

## 4. 🎫 免失败运输卡 怎么用

- 平时拿到的卡显示在卡片顶部右上 (🎫×N)
- 卖东西时, 销售 modal 会出现"🎫 使用免失败运输卡"勾选框
- **勾选 → 跳过骰子 → 100% 成功送达** (但卡会消耗 1 张)
- 适合用在**高价值/大数量**的运输 (一旦失败损失大)

## 5. 📈 升级 (Level Up) + 🏆 终极完成

资产/进度达标后, 国家可以**升级**,获得更高威望。升级门槛在 Google Sheet **LevelUp** 表里设置,导师可随时改。

> ⚠️ **关于 Lv.4 终极完成:** Lv.4 是最高等级 (`level` 字段最大为 4),表里 "🏆 Lv.4 终极完成" 那一行是**赢游戏的最终条件**。达成这些**不会让 level 变成 5**,只会让卡片显示「🏆 终极完成 ✓」徽章。最终按总资产 + 终极完成状态排胜负。

| 升级 / 目标 | 需要 |
|---|---|
| Lv.1 → Lv.2 | 完成 L1 产品 ≥ 10, 金币 ≥ 20,000, 成功运输 ≥ 5 |
| Lv.2 → Lv.3 | 完成 L2 产品 ≥ 8, 金币 ≥ 50,000 |
| Lv.3 → Lv.4 | 完成 L3 产品 ≥ 5, 金币 ≥ 90,000 |
| 🏆 Lv.4 终极完成 | 不同 L4 产品 ≥ 2, 金币 ≥ 120,000, 爱心值 ≥ 1, 荣誉值 ≥ 30 |

## 6. 💎 总资产怎么算

```
总资产 = ((金币 + 产品估值) × 爱心倍数) + (荣誉值 × 1000)
```

**公式的四个组成部分:**

| 项 | 含义 |
|---|---|
| 💰 **金币** | 你账上的钱 |
| 📦 **产品估值** | 库存里每个 L1/L2/L3/L4 产品的 `asset_value` 之和 (资源估值默认 0,不算) |
| ❤️ **爱心倍数** | 见下方 LoveTable,作用于「金币+产品估值」(乘法) |
| 🏅 **荣誉值** | 每点 = +1,000 金币加成 (不进倍数,直接相加) |

**❤️ 爱心倍数表 (LoveTable · sheet 实时):**

| ❤️ 爱心 | 倍数 | 说明 |
|---|---|---|
| 0 | 1.00 | 基础 · 无加成 |
| 1 | 1.10 | +10% |
| 2 | 1.12 | +12% |
| 3 | 1.15 | +15% |
| 4 | 1.18 | +18% |
| 5 | 1.20 | +20% |
| 6 | 1.22 | +22% |
| 7 | 1.25 | +25% |
| 8 | 1.28 | +28% |
| 9 | 1.29 | +29% |
| 10 | 1.30 | +30% |
| 11 | 1.31 | +31% |
| 12 | 1.32 | +32% |
| 13 | 1.33 | +33% |
| 14 | 1.34 | +34% |
| 15 | 1.35 | +35% |

> 💡 **爱心 vs 荣誉的差别:** 爱心是**乘法**(放大整个金币+产品估值池),荣誉是**加法**(每点固定 +1,000)。爱心对富国效果指数级放大;荣誉对穷国相对更划算。

## 7. 🚦 团队策略建议

### 🇲🇾 马来西亚 (资源国)
- 起手 400 ×6 = 2400 单位资源, **够你免费做 ~12 个 L1**
- 早期囤一堆 L1, 卖给银行换金币 + 升级条件
- 中期开始把资源**卖给别国**赚高溢价 (尤其 JP 没自己的资源)

### 🇰🇷 韩国 (科技国)
- **L1 卖银行多 15%** (sell_price 2300 vs 标准 2000), 早期开仓做 L1 卖银行回血金币最快
- 然后升级到 L2/L3, 走电子/AI 高端路线
- 缺资源 → 跟 MY 买

### 🇯🇵 日本 (医疗国)
- 买资源**只要 450**, 比别人省 10%, 长期累积优势大
- 主打医疗产品 (L1 医疗用品/药品包/防护装备 → L2 医疗设备/实验仪器)
- L3 医疗 AI 诊断系统 + L4 全球医疗网络是高溢价目标

### 🇺🇸 美国 (金融国)
- 起手金币 20000, **可以暴买半成品/资源跳级**
- 主打金融/国防/太空,L3/L4 产品价值最高
- 但要小心: 钱多容易乱花,先想清楚配方再买

## 8. ⚠️ 常见错误

| 症状 | 原因 | 解决 |
|---|---|---|
| 制造按钮点进去空空的 | 没采购单 | 先去 📋 订单申请 |
| 申请订单"金币不足" | 没 100 金币 | 卖东西回血 |
| 申请订单"已达上限" | 当前国家等级的持单上限已满 (Lv.1=2, Lv.2=3, Lv.3+=5) | 先生产消耗掉一张, 或升级国家 |
| 制造提示"资源不足" | 配方里的某种资源不够 | 去 🛒 采购 或跟别国买 |
| 制造提示"半成品不足" | L2+ 需要 L1 半成品 | 先做 L1, 或买别国的 |
| 销售提示"买方金币不足" | 对方国家没钱 | 谈低价或卖给银行 |

## 9. 🏁 终局

游戏结束时,**资产最高**的国家胜出,但**爱心/荣誉**也是评分维度。

- 单纯卷资产 → 容易忽略爱心 (协作/关怀的得分)
- 单纯做好人 → 资产可能跑不过别国
- **平衡发展** — 爱心 (×倍数) 和荣誉 (+加成) 都影响最终总资产排名
